સ્માર્ટ દસ્તાવેજ પ્રક્રિયા પોર્ટલ

PDF વિભાજન અને એક્સપોર્ટ ટૂલ

તમારી PDF અને Excel ફાઇલ અહીં અપલોડ કરો. Excel માં આપેલ નિયમો મુજબ સિસ્ટમ આપમેળે ફાઇલને પ્રોસેસ કરીને ZIP ફાઇલ બનાવશે.

ફાઇલ ક્યાં અપલોડ કરવી?
નીચે આપેલ પ્રથમ બોક્સમાં PDF ફાઇલ અપલોડ કરવી.
બીજા બોક્સમાં Excel ફાઇલ અપલોડ કરવી.
પછી “પ્રોસેસ કરો અને ZIP ડાઉનલોડ કરો” બટન દબાવવું.

નમૂના ફાઇલ ડાઉનલોડ કરો

વપરાશકર્તાને સરળતા રહે તે માટે નીચે નમૂના PDF અને Excel ફાઇલની લિંક આપવામાં આવી છે. અહીં તમારી સાચી લિંક મૂકી શકો છો.

નમૂના PDF ડાઉનલોડ કરો નમૂના Excel ડાઉનલોડ કરો
અહીં PDF ફાઇલ અપલોડ કરો
મુખ્ય PDF ફાઇલ અહીં પસંદ કરો. આ ફાઇલને Excel નિયમો પ્રમાણે ભાગોમાં વહેંચવામાં આવશે.
અહીં Excel ફાઇલ અપલોડ કરો
Excel શીટમાં પેજ રેન્જ, આઉટપુટ ફોર્મેટ અને ફાઇલ નામ આપેલું હોવું જોઈએ.

Excel ફોર્મેટ કેવી રીતે હોવું જોઈએ:
કોલમ A = પેજ રેન્જ
કોલમ B = આઉટપુટ ફોર્મેટ જેમ કે pdf / jpg
કોલમ C = આઉટપુટ ફાઇલ નામ

આ ટૂલ કેવી રીતે મદદરૂપ છે

આ પોર્ટલ ઝડપી દસ્તાવેજ પ્રક્રિયા, સરળ અપલોડ, સ્વચ્છ આઉટપુટ અને ઓફિસના દૈનિક ઉપયોગ માટે ખૂબ જ ઉપયોગી રીતે બનાવવામાં આવ્યું છે.

ઝડપી પ્રોસેસિંગ Excel સૂચનાઓ પ્રમાણે એક જ ક્લિકમાં PDF પેજોને અલગ કરો.
સરળ ઉપયોગ કોઈપણ કર્મચારી સરળતાથી સમજીને ઉપયોગ કરી શકે તેવી ડિઝાઇન.
તૈયાર ZIP આઉટપુટ બધી બનાવેલી ફાઇલો આપમેળે ZIPમાં તૈયાર થઈને ડાઉનલોડ માટે મળે છે.
